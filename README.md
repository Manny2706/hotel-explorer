# 🚀 Features

- 🔍 Search hotels by name
- 📍 Filter by location
- ⭐ Filter by minimum rating
- 💰 Sort by price and rating
- 📄 Detailed hotel page
- 📱 Fully responsive UI
- ⚡ Fast API integration using Axios
- 🗂 Global state management with Redux Toolkit
- 🎨 Premium UI built with Tailwind CSS
- 🔄 Pagination support
- ⏳ Skeleton loading states
- ❌ Empty & error states

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/hotel-finder.git
```

Move into the project

```bash
cd hotel-finder
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build project

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
